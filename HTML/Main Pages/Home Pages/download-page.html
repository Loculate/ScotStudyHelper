1. Ensure you are currently using an android device, as the application is only supported on the android framework.

2. In the device settings, ensure that 'allow downloads from unknown sources' is enabled.

3. Download the ADK file from the link below.

4. Navigate to the device's files usually stored under 'files' and open the ADK file previously downloaded.
